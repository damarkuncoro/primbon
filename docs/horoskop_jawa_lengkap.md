# Horoskop Jawa Lengkap: Pranata Mangsa

Sistem astrologi Jawa yang kaya, mencakup 12 mangsa yang masing-masing memiliki karakteristik unik.

## Struktur Pranata Mangsa

1. **Kasa** (23 Juni - 2 Agustus)
2. **Karo** (3 Agustus - 25 Agustus)
3. **Katelu** (26 Agustus - 18 September)
4. **Kapat** (19 September - 12 Oktober)
5. **Kalima** (13 Oktober - 9 November)
6. **Kanem** (10 November - 22 Desember)
7. **Kapitu** (23 Desember - 3 Februari)
8. **Kawolu** (4 Februari - 1 Maret)
9. **Kasanga** (2 Maret - 25 Maret)
10. **Kasadasa** (26 Maret - 18 April)
11. **Dhesta** (19 April - 11 Mei)
12. **Saddha** (12 Mei - 22 Juni)

## Keadaan Kesehatan Berdasarkan Mangsa

| Mangsa | Karakteristik Kesehatan |
| :--- | :--- |
| **Kasa** | Kuat, waspada terhadap jantung/tekanan darah. |
| **Karo** | Sensitif terhadap lingkungan, alergi/pernapasan. |
| **Katelu** | Metabolisme cepat, perhatikan pencernaan. |
| **Kapat** | Kesehatan stabil, fokus pada kesehatan mental. |
| **Kalima** | Kesehatan baik, waspada cedera tulang/sendi. |
| **Kanem** | Rentan terhadap sistem kekebalan tubuh. |
| **Kapitu** | Kesehatan mental baik, waspada masalah tidur. |
| **Kawolu** | Energi tinggi, waspada sistem saraf. |
| **Kasanga** | Stabil, perhatikan sistem peredaran darah. |
| **Kasadasa** | Kesehatan emosional baik, perhatikan hormon. |
| **Dhesta** | Energi tinggi, perhatikan kolesterol. |
| **Saddha** | Stabil, perhatikan sistem pencernaan. |

## Kecocokan Karir

Setiap mangsa memiliki kecenderungan pekerjaan tertentu:
- **Kasa**: Pemimpin, Manajer.
- **Karo**: Seniman, Desainer.
- **Katelu**: Akuntan, Insinyur.
- **Kapat**: Mediator, Hakim.
- **Kalima**: Atlet, Tentara, Kontraktor.
- **Kanem**: Pekerja Sosial, Guru.
- **Kapitu**: Penulis, Filsuf.
- **Kawolu**: Pengusaha, Pilot.
- **Kasanga**: Arsitek, Manajer Proyek.
- **Kasadasa**: Psikolog, Konselor.
- **Dhesta**: Politisi, Pengacara.
- **Saddha**: Petani, Peneliti.
