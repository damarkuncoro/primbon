# Deskripsi Pengembangan

## NPM Package: `@damarkuncoro/primbon`

Library JavaScript berbasis Node.js yang menyediakan **algoritma kalender Jawa, weton, primbon, dan perhitungan tradisional Jawa** untuk digunakan dalam aplikasi web, mobile, maupun backend.

Package ini bertujuan menjadi **library budaya Jawa paling lengkap di ekosistem JavaScript**.

---

# 1. Tujuan Pengembangan

Membangun library yang mampu:

1. Menghitung **weton dari tanggal masehi**
2. Menghitung **pasaran Jawa**
3. Menghitung **neptu weton**
4. Menentukan **watak berdasarkan weton**
5. Menghitung **kecocokan jodoh**
6. Menentukan **hari baik**
7. Menghasilkan **kalender Jawa lengkap**
8. Mendukung **pawukon dan wuku Jawa**
9. Mendukung **siklus 35 hari weton**
10. Digunakan di berbagai framework modern

Contoh penggunaan:

* aplikasi kalender Jawa
* aplikasi spiritual / primbon
* game bertema Nusantara
* AI budaya
* chatbot budaya Jawa
* website budaya

# 4. Modul Utama

## 4.1 Modul Kalender Jawa

Mengkonversi **tanggal masehi → sistem kalender Jawa**

Output:

```
{
 hari: "Jumat",
 pasaran: "Kliwon",
 weton: "Jumat Kliwon",
 neptu: 14
}
```

Fungsi:

```
primbon.weton(tanggal)
```

---

# 4.2 Modul Pasaran Jawa

Pasaran adalah siklus **5 hari**:

* Legi
* Pahing
* Pon
* Wage
* Kliwon

Algoritma:

```
pasaran = (selisihHari % 5)
```

Fungsi:

```
primbon.pasaran(tanggal)
```

---

# 4.3 Modul Neptu

Menghitung nilai weton.

Tabel neptu:

Hari:

```
Minggu = 5
Senin = 4
Selasa = 3
Rabu = 7
Kamis = 8
Jumat = 6
Sabtu = 9
```

Pasaran:

```
Legi = 5
Pahing = 9
Pon = 7
Wage = 4
Kliwon = 8
```

Fungsi:

```
primbon.neptu("Jumat","Kliwon")
```

Output:

```
14
```

---

# 4.4 Modul Watak Weton

Setiap weton memiliki karakter.

Contoh:

```
Jumat Kliwon:
- spiritual kuat
- bijaksana
- pemimpin alami
- sensitif energi
```

Fungsi:

```
primbon.watak(tanggal)
```

---

# 4.5 Modul Jodoh Weton

Perhitungan kecocokan berdasarkan jumlah **neptu pasangan**.

Rumus:

```
(neptuA + neptuB) % 8
```

Kategori hasil:

```
Pegat
Ratu
Jodoh
Topo
Tinari
Padu
Sujanan
Pesthi
```

Fungsi:

```
primbon.jodoh(tglA, tglB)
```

Output:

```
{
 kategori: "Ratu",
 arti: "Pasangan dihormati masyarakat"
}
```

---

# 4.6 Modul Hari Baik

Digunakan untuk menentukan hari baik untuk:

* nikah
* pindah rumah
* membuka usaha
* perjalanan

Fungsi:

```
primbon.hariBaik("nikah", bulan)
```

Output:

```
[
 "Jumat Kliwon",
 "Sabtu Legi",
 "Rabu Pon"
]
```

---

# 4.7 Modul Pawukon

Pawukon adalah kalender Jawa **210 hari**.

Terdiri dari **30 wuku**:

Contoh:

```
Sinta
Landep
Wukir
Kurantil
Tolu
Gumbreg
Warigalit
Warigagung
Julungwangi
Sungsang
```

Fungsi:

```
primbon.wuku(tanggal)
```

Output:

```
Wuku: Landep
```

# 6. API Library

Contoh penggunaan:

```
import primbon from "@nusantara/primbon"

primbon.weton("1990-08-17")
```

Output:

```
{
 hari: "Jumat",
 pasaran: "Kliwon",
 weton: "Jumat Kliwon",
 neptu: 14
}
```

---

Contoh lain:

```
primbon.jodoh("1990-08-17","1995-01-10")
```

Output:

```
{
 kategori: "Tinari",
 arti: "Rezeki dan keberuntungan"
}
```
