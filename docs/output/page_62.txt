Jika pria tidak banyak bicara, jika wanita banyak
bicara tanpa memandang pria.

Kokoh pendapatnya, pandai mencari harta serta
kaya, dermawan, sebaiknya memelihara kuda.
Nampaknya angkuh, pandai, banyak akalnya, berhati-
hati, perasaannya halus (pandai memutar dunia)
Manis pembicaraannya, pandai mengambil hati
orang lain dan dapat mencari rejeki.

Tidak suka mengindahkan nasehat, pembicaraan-
nya sulit diatur, tetapi jika wanita baik, dapat
menyimpan uang.

Pandai berbicara, fikirannya selalu tertuju pada wa-
nita, sulit bersikap seperti pendeta (rusuh).

Luas pandangan/fikirannya, tidak suka diatasi.
Jika wanita : jelek, sering bertengkar, tidak meng-
hormati pria.

Pendiam, jika bodoh terlampau sekali tetapi jika
pandai, pandai sekali. Menjadi teladan banyak orang
dan banyak memiliki pengetahuan.
