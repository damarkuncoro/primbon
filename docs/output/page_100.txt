Watak wanita menurut hari kelahiran.

Jika lahir pada hari Sabtu Paing maka akan ada baiknya serta
buruknya, yaitu : jika baik : sangat berlebihan kebaikannya, jika
buruk: sangat berlebihan juga buruknya, selalu merasa tidak puas
terhadap suami.

Jika lahir pada hari Ahad Pon, maka akan berbakti terhadap
suami, keras hatinya, cemburuan.

Jika lahir pada hari Senin Wage maka akan berbakti terhadap
suami, pandai, keras dan pandai berbicara, tidak suka jika terka-
lahkan suami.

Jika lahir pada hari Selasa Kliwon: maka tidak akan berbakti pada
suami dan mudah tergoda.

Jika lahir pada hari Rabu Legi : maka akan ada baiknya dan
buruknya yaitu : jika baik: kebaikannya sangat berlebihan,jika
buruk: sangat berlebihan buruknya.

Jika lahir pada hari Kamis Paing: maka akan berbakti terhadap
suami tetapi mudah tergoda.

Jika lahir pada hari Jum'at Pon: akan berani menyangkal
perkataan suami,pemboros, banyak mendapatkan rintangan akan
tetapi selalu memperoleh anugerah dari Tuhan dan segala keingi-
nannya terkabul.

Jika lahir pada hari Sabtu Wage: maka akan lebih berbakti
terhadap suami, pandai, teliti, berhati-hati, akan tetapi sering
kematian anaknya.

Jika lahir pada hari Ahad Kliwon : wanita tersebut : pemalas,
cemburuan, kurang merawat suami.

Jika lahir pada hari Senin Legi : akan berani terhadap suami, tetapi
membawa rejeki.

Jika lahir pada hari Selasa Paing : wanita ini kurang merawat
suami dan suka memperhatikan dirinya.

Jika lahir pada hari Jum'at Kliwon: berbakti terhadap suami,

Jika lahir pada hari Sabtu Legi.: memiliki rejeki, berani membantah

dan berani terhadap suami.

Jika lahir pada hari Senin Pon: terhadap suami tidak pernah
