No. 206. Makna hari dan pekan / pasaran.

Perhitungannya menurut jumlah neptu hari dan pekan /
pasaran. Jika hasil penjumlahan tersebut :

Neptu 7, 11, 15, : Lambangnya Janggleng, maknanya : jika ber-
perang terasa lambat, sering kembali, akan tidak
berhasil jika menagih ataupun berhutang.

Neptu 8, 12, 16 : Lambangnya Celeng, maknanya : jika berperang
bingung, akan tidak berhasill jika menagih atau
pun berhutang. ,

Neptu 9, 13, 17, : Lambangnya Nyangking, maknanya : jika ber-
perang akan dapat menyelesaikannya, akan mu-
dah berhasil menagih ataupun berhutang.

Neptu 10, 14, 18 : Lambangnya Kithing, maknanya : tak akan terja-
di jika mau berperang, akan tidak berhasil jika
menagih ataupun berhutang.
